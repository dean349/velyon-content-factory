# Velyon Portfolio Forge — Product Description

## Tagline
An adaptable AI content pipeline that discovers, curates, and generates case studies and marketing assets from any organisation's deployed portfolio.

## Description
A three-phase platform (Discover → Curate → Generate) that scans Vercel, GitHub, Netlify, Cloudflare, and live URLs for deployed projects, classifies them with AI taxonomy tagging, and batch-produces publication-ready case studies, NotebookLM exports, video scripts, and CMS assets. Dual-catalog architecture for client work and proprietary products, with per-item visibility controls, cross-linking, and multi-format export — deployable as a bespoke instance for any business with a portfolio to showcase.

## Long Description

### Context
Every technology company, agency, and consultancy builds work that could become a case study — but most lack the system to turn deployed projects into polished marketing content. Projects live across hosting platforms, repos stay disconnected from marketing, and the gap between "we built this" and "here's proof we built this" costs businesses deals.

### Approach
Content Factory is a custom-deployable pipeline designed to plug into any organisation's infrastructure. Discovery scans Vercel, GitHub, Netlify, Cloudflare, and arbitrary URLs to pull in every deployed project as a raw item. Curation provides a dual-catalog editor — Case Studies for client work, Products for proprietary IP — with structured tabs for metrics, tech stack, testimonials, team, redaction rules, and AI-powered classification. Cross-linking connects case studies to the products that powered them. Generation batch-produces output across markdown, NotebookLM JSON, cinematic video scripts, PDF, CMS, and social exports. Visibility controls (Public / Client Approved / Redacted / Internal Only) ensure sensitive client work never leaks into public output while remaining trackable internally.

### Outcome
Organisations using the platform can classify an entire portfolio with a single click, batch-generate case studies from curated entries, and export content across multiple formats without rebuilding narratives project by project. The adaptable architecture means it ships under the client's brand, connects to their existing auth systems, and scales from a 10-project boutique to a 500-project enterprise portfolio.

### What's Next
Being extended with bulk auto-classification, NotebookLM audio/video overview generation, and AI avatar video integration. The dual-catalog architecture also enables a public-facing portfolio page that auto-populates from curated entries — turning Content Factory into both a content engine and a live showcase for any business.
